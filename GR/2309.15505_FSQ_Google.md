## 核心贡献

本文提出 **FSQ（Finite Scalar Quantization，有限标量量化）**，作为 VQ-VAE 中向量量化（VQ）的简单替代方案。

**核心思路**：将 VAE 的表示投影到极少维度（通常 $d < 10$），对每个维度独立进行有界整数量化，从而隐式定义一个码本，大小为各维度量化级数之积 $|\mathcal{C}| = \prod_{i=1}^d L_i$。

---

## 背景：VQ-VAE 的问题

VQ-VAE 中的向量量化面临以下已知挑战：

| 问题 | 描述 |
|------|------|
| **码本坍塌（Codebook Collapse）** | 随码本规模增大，大量码字从未被使用 |
| **优化复杂** | 需要 commitment loss、codebook loss、entropy loss 等多个辅助损失 |
| **工程技巧繁多** | EMA 更新码本、码本重初始化（reseeding）、码本分裂（splitting）、熵惩罚等 |
| **参数量大** | 典型 $\mathcal{C}=4096, d=512$ 时，码本本身有 2M 参数 |

---

## FSQ 方法详解

### 量化过程

给定 $d$ 维表示 $z \in \mathbb{R}^d$：

1. **Bound（有界映射）**：对每个通道 $z_i$ 应用有界函数 $f$，映射到 $L_i$ 个离散整数值
   $$f: z_i \mapsto \lfloor L_i/2 \rfloor \cdot \tanh(z_i + \text{offset}) - \text{offset}$$

2. **Round（取整）**：对 $f(z)$ 取整，得到量化表示 $\hat{z} = \text{round}(f(z))$

3. **隐式码本**：$\hat{z}$ 必然是 $\mathcal{C} = \{L_1\} \times \{L_2\} \times \cdots \times \{L_d\}$ 中的某个元素，码本大小 $|\mathcal{C}| = \prod_i L_i$

4. **梯度传播**：使用 Straight-Through Estimator（STE），将 round 的梯度替换为 1：
   $$\texttt{round-ste}: x \mapsto x + \text{sg}(\text{round}(x) - x)$$

### FSQ vs VQ 对比

| 属性 | VQ | FSQ |
|------|-----|-----|
| 量化方式 | $\arg\min_{c \in \mathcal{C}} \|z - c\|$（最近邻查找） | $\text{round}(f(z))$（独立逐维取整） |
| 梯度 | STE | STE |
| 辅助损失 | Commitment loss + Codebook loss + Entropy loss | **无** |
| 工程技巧 | EMA、码本分裂、重初始化等 | **无** |
| 码本参数 | 有（$|\mathcal{C}| \times d$ 个参数） | **无（隐式码本）** |
| 典型维度 | $d \ge 512$ | $d < 10$ |

### 推荐超参数配置

| 目标码本大小 | 推荐 $\mathcal{L}$ |
|------------|-----------------|
| $2^8 = 256$ | $[8, 6, 5]$ |
| $2^{10} = 1024$ | $[8, 5, 5, 5]$ |
| $2^{12} = 4096$ | $[7, 5, 5, 5, 5]$ |
| $2^{14} = 16384$ | $[8, 8, 8, 6, 5]$ |
| $2^{16} = 65536$ | $[8, 8, 8, 5, 5, 5]$ |

**关键启发式规则**：每维量化级数 $L_i \ge 5$ 时效果最佳。

## 实验设置

论文在两个主要模型框架上验证 FSQ：

### 1. MaskGIT（图像生成）
- **Stage I**：训练卷积 VQ-GAN 自编码器（重建）
- **Stage II**：在量化表示上训练 Masked Transformer（BERT 风格）进行图像生成
- **数据集**：ImageNet 256×256
- **FSQ 配置**：$\mathcal{L} = [8, 5, 5, 5]$（对应 $|\mathcal{C}| = 2^{10}$）

### 2. UViM（密集预测）
- **任务**：深度估计（NYU Depth v2）、全景分割（COCO Panoptic）、图像着色（ImageNet）
- **架构**：Transformer-based VQ-VAE + Encoder-Decoder Transformer
- **FSQ 配置**：$\mathcal{L} = [7, 5, 5, 5, 5]$（对应 $|\mathcal{C}| = 2^{12}$）

## 实验结果

### MaskGIT（ImageNet 256×256）

| 模型 | Sampling FID↓ | Precision↑ | Recall↑ | 码本使用率↑ |
|------|-------------|-----------|--------|----------|
| MaskGIT (VQ) | 4.509 | 0.860 | 0.465 | 81% |
| **MaskGIT (FSQ)** | **4.534** | **0.864** | **0.453** | **100%** |
| MaskGIT (VQ, 官方) | 4.916 | 0.836 | 0.489 | - |
| ADM（扩散模型） | 4.59 | 0.83 | 0.52 | - |

**结论**：FSQ 与 VQ 性能几乎相当（FID 差距 < 0.03），但码本使用率达到 100%（VQ 仅 81%）。

### UViM（密集预测）

| 任务 | 指标 | VQ（我们） | FSQ（我们） | VQ（官方 GitHub） |
|------|------|-----------|-----------|----------------|
| 深度估计 | RMSE↓ | 0.468 ± 0.012 | 0.473 ± 0.012 | 0.463 |
| 全景分割 | PQ↑ | 43.4 | 43.2 | 43.1 |
| 图像着色 | FID-5k↓ | 16.90 | 17.55 | 16.99 |

**结论**：FSQ 在所有任务上均具有竞争力，性能仅略低于 VQ（0.5%~3%），但无需任何辅助技巧。

### 码本规模扩展性研究（128×128 ImageNet）

| 现象 | 描述 |
|------|------|
| **FSQ 重建 FID 随码本规模单调递减** | 更大码本 → 更多信息存储 → 更好重建 |
| **VQ 重建 FID 在 $2^{11}$ 后变差** | 码本坍塌导致大码本无法有效利用 |
| **FSQ 码本使用率接近 100%** | $2^{14}$ 时仍 ≈100%，VQ 在 $2^{11}$ 后急剧下降 |
| **FSQ 采样 FID 在 $2^{12}$ 后趋于饱和** | 更大码本导致 Transformer 建模难度增加 |

## 关键分析

### 码本分裂消融（UViM 深度估计）

| 模型 | RMSE↓ | 码本使用率 |
|------|-------|---------|
| UViM (VQ) | 0.468 | 99% |

### 无上下文信息消融（UViM 全景分割）

|------|-----|
| UViM (VQ, with context) | 43.4 |
| UViM (FSQ, with context) | 43.2 |

FSQ 在无侧信息时性能下降更少，鲁棒性更强。

## 表示语义分析

论文对 FSQ 和 VQ 码本的语义性进行了研究（图像拼接实验、随机码分布采样）：
- **两者均未发现固定的语义码字**：单个码字不代表固定的视觉概念
- **解码器权重决定最终图像**：语义来自码组合 + 解码器，而非单一码字
- FSQ 的隐式码本与 VQ 的显式学习码本在表示能力上无显著差异

## 核心代码（JAX 实现）

```python
def round_ste(z):
    zhat = jnp.round(z)
    return z + jax.lax.stop_gradient(zhat - z)

class FSQ:
    def __init__(self, levels: list[int]):
        self._levels = levels
        self._levels_np = np.asarray(levels)
        self._basis = np.concatenate(
            ([1], np.cumprod(self._levels_np[:-1]))
        ).astype(np.uint32)

    def bound(self, z):
        eps = 1e-3
        half_l = (self._levels_np - 1) * (1 - eps) / 2
        offset = jnp.where(self._levels_np % 2 == 1, 0.0, 0.5)
        shift = jnp.tan(offset / half_l)
        return jnp.tanh(z + shift) * half_l - offset

    def quantize(self, z):
        quantized = round_ste(self.bound(z))
        half_width = self._levels_np // 2
        return quantized / half_width
```

## 总结

| 维度 | 评价 |
|------|------|
| **创新性** | 将神经压缩领域的有界标量量化引入表示学习，视角新颖 |
| **简洁性** | 彻底消除 VQ 的所有辅助损失和工程技巧，实现极简 |
| **性能** | 与 VQ 性能相当（差距 0.5%~3%），但在大码本场景下更优 |
| **码本利用率** | 几乎 100%，VQ 在大码本下严重坍塌 |
| **扩展性** | 随码本规模扩大，FSQ 持续受益；VQ 则出现性能退化 |
| **局限性** | FSQ 的隐式码本维度极低（$d<10$），表达能力理论上不如高维 VQ 码本 |

**一句话总结**：FSQ 用"把表示投影到几个维度后逐维取整"这一极简操作，替代了 VQ-VAE 中复杂的向量量化机制，在保持相当性能的同时，彻底解决了码本坍塌问题，并消除了所有辅助损失和工程技巧。
