## 研究背景与动机

### 什么是语义 ID（Semantic ID, SID）？

语义 ID 是从多模态 item 特征（文本、图像、音频、结构化元数据）中提炼出的紧凑离散表示，可作为 ID-based 推荐和生成式推荐的统一抽象接口。相比传统 ID embedding，SID 具有以下优势：
- 避免 embedding 表的无限增长（vocabulary explosion）
- 减少哈希碰撞噪声
- 在 item 频繁更新、重新索引时保持稳定

主流范式是基于 **Residual Quantized VAE（RQ-VAE）** 将多模态特征层次化量化为多个 codebook 索引序列。

### 现有方法的两大核心问题

**问题一：离散空间中的碰撞（Collision Problem）**

大量 item 被压缩到有限的量化码本时，语义上不相关的 item 往往被分配到相同或高度相似的 SID 组合，造成"语义纠缠"（semantic entanglement），下游模型难以区分本质不同的 item。

**问题二：碰撞信号的异质性（Collision-Signal Heterogeneity）**

并非所有碰撞都有害：
- **有害碰撞**：语义无关 item 被分配了相同/相近的 SID → 需要排斥
- **良性重叠**：同一 item 的重复采样、训练流水线中有意构建的协同正样本对 → 不应排斥

现有方法（如 SaviorRec、LETTER、HiD-VAE）对所有碰撞一视同仁地抑制，可能错误地推开良性对，干扰下游对齐。

## 核心方法：QuaSID

**QuaSID（Qualification-Aware Semantic ID Learning）** 是一个端到端 SID 学习框架，核心思想是：**只对"有资质"的冲突对施加排斥，并按碰撞严重程度缩放排斥强度**。

### 整体框架

QuaSID 包含以下四个模块：

```
多模态输入 x
    ↓ 编码器 f_θ
连续嵌入 z
    ↓ L 层残差向量量化
量化表示 ẑ + SID s = [s^(1), ..., s^(L)]
    ↓ 解码器 h_φ
重建输出 x̂
```

同时引入：
1. **HaMR**：Hamming-guided Margin Repulsion（汉明引导的边界排斥）
2. **CVPM**：Conflict-Aware Valid Pair Masking（冲突感知有效对掩码）
3. **双塔对比学习**：注入协同语义信号

### 符号定义

- $\mathcal{U}, \mathcal{I}$：用户集合和 item 集合
- $\mathbf{x}_i \in \mathbb{R}^{d_{\text{in}}}$：item $i$ 的多模态特征向量
- $\mathbf{z}_i = f_\theta(\mathbf{x}_i) \in \mathbb{R}^d$：连续嵌入
- $\mathcal{C}^{(l)} = \{\mathbf{c}^{(l)}_k\}_{k=1}^K$：第 $l$ 层码本（共 $L$ 层，每层 $K$ 个码字）
- $\mathbf{s}_i = [s_i^{(1)}, s_i^{(2)}, \ldots, s_i^{(L)}] \in \{1,\ldots,K\}^L$：item $i$ 的 SID

### 模块一：Tokenizer Backbone（RQ-VAE）

残差量化过程：

$$\mathbf{q}_i^{(l)} = \mathbf{c}^{(l)}_{s_i^{(l)}}, \quad \mathbf{r}_i^{(l)} = \mathbf{r}_i^{(l-1)} - \mathbf{q}_i^{(l)}, \quad l=1,\ldots,L$$

$$\hat{\mathbf{z}}_i = \sum_{l=1}^{L}\mathbf{q}_i^{(l)}, \quad \mathbf{s}_i = [s_i^{(1)},\ldots,s_i^{(L)}]$$

损失函数：

$$\mathcal{L}_{\text{rec}} = \frac{1}{|\mathcal{B}|}\sum_{i\in\mathcal{B}} \ell_{\text{rec}}(\hat{\mathbf{x}}_i, \mathbf{x}_i)$$

$$\mathcal{L}_{\text{rq}} = \frac{1}{|\mathcal{B}|}\sum_{i\in\mathcal{B}}\sum_{l=1}^{L} \left(\|\mathrm{sg}[\mathbf{r}_i^{(l-1)}]-\mathbf{q}_i^{(l)}\|_2^2 + \beta\|\mathbf{r}_i^{(l-1)}-\mathrm{sg}[\mathbf{q}_i^{(l)}]\|_2^2\right)$$

其中 $\mathrm{sg}[\cdot]$ 为 stop-gradient，$\beta$ 控制 commitment 强度。

### 模块二：协同双塔对比学习

基于 item-item 协同对（trigger-target pairs，通过 Swing 算法构建），使用 InfoNCE 损失注入协同语义：

$$\mathcal{L}_{\text{cl}} = -\frac{\lambda_{\text{cl}}}{B}\sum_{m=1}^{B}\log\frac{\exp(\mathbf{S}_{m,m})}{\sum_{n=1}^{B}\mathbb{I}[M_{m,n}=1]\exp(\mathbf{S}_{m,n})}$$

其中 $\mathbf{S}_{m,n} = (\mathbf{e}_t^m)^\top \mathbf{e}_p^n / \tau$，同 item ID 的负样本被掩码以减少假负例偏差。

### 模块三：CVPM（冲突感知有效对掩码）

在 mini-batch 中构建 $2B$ 个实例（trigger + target），通过两个掩码过滤良性重叠：

**（1）协同正样本掩码**（排除对比学习中的正样本对，避免排斥与任务对齐矛盾）：

$$\mathbf{M}^{\text{i2i}}_{ij} = \begin{cases} 0, & (i \le B \wedge j=i+B) \lor (j \le B \wedge i=j+B) \\ 1, & \text{otherwise} \end{cases}$$

**（2）同 item 排除掩码**（排除同一 item 的重复采样）：

$$\mathbf{M}^{\text{item}}_{ij} = \mathbb{I}[\mathrm{id}(i) \neq \mathrm{id}(j)]$$

**组合掩码**：

$$\mathbf{M} = \mathbf{M}^{\text{i2i}} \odot \mathbf{M}^{\text{item}}$$

### 模块四：HaMR（汉明引导边界排斥）

定义 pairwise 汉明距离矩阵 $\mathbf{H}$ 和余弦距离矩阵 $\mathbf{D}$：

$$\mathbf{H}_{ij} = d_H(i,j) = \sum_{l=1}^{L}\mathbb{I}[s_i^{(l)} \neq s_j^{(l)}]$$

$$\mathbf{D}_{ij} = d_c(i,j) = 1 - \mathbf{e}_i^\top\mathbf{e}_j$$

将碰撞分为两类（$R$ 为汉明半径超参数）：

- **完全碰撞集合**：$\Omega_{\text{full}} = \{(i,j) \mid \mathbf{H}_{ij}=0 \text{ and } \mathbf{M}_{ij}=1\}$
- **部分碰撞集合**：$\Omega_{\text{partial}} = \{(i,j) \mid 0 < \mathbf{H}_{ij} \le R \text{ and } \mathbf{M}_{ij}=1\}$

对每对施加 hinge 损失（$m_{\text{full}} \ge m_{\text{partial}}$，完全碰撞惩罚更强）：

$$\mathcal{L}_{\text{full}}(i,j) = \max(0,\ m_{\text{full}} - \mathbf{D}_{ij})$$

$$\mathcal{L}_{\text{partial}}(i,j) = \max(0,\ m_{\text{partial}} - \mathbf{D}_{ij})$$

$$\mathcal{L}_{\text{HaMR}} = \frac{\lambda_{\text{full}}}{|\Omega_{\text{full}}|+\epsilon}\sum_{(i,j)\in\Omega_{\text{full}}}\mathcal{L}_{\text{full}}(i,j) + \frac{\lambda_{\text{partial}}}{|\Omega_{\text{partial}}|+\epsilon}\sum_{(i,j)\in\Omega_{\text{partial}}}\mathcal{L}_{\text{partial}}(i,j)$$

### 总体优化目标

$$\mathcal{L} = \mathcal{L}_{\text{rec}} + \mathcal{L}_{\text{rq}} + \mathcal{L}_{\text{HaMR}} + \mathcal{L}_{\text{cl}}$$

所有组件端到端训练，离散码分配通过 straight-through estimator（STE）实现可微学习。

## 实验设置

### 数据集

| 数据集 | 用户数 | item 数 | 交互数 |
|--------|--------|---------|--------|
| Amazon-Beauty | 22,363 | 12,101 | 1,048,296 |
| Amazon-Toys | 19,412 | 11,924 | 905,253 |

- 使用 5-core 过滤，leave-one-out 划分
- item 文本特征（Title/Brand/Categories/Price）通过 Sentence-T5-XXL 提取语义嵌入
- **工业数据集**：快手电商平台，item 包含文本描述、ASR 转录、关键帧图像，通过多模态大语言模型提取嵌入

### 基线方法

| 方法 | 核心特点 |
|------|---------|
| RQ-VAE | 标准残差量化 VAE |
| GRVQ | 分组残差量化，减少跨维度干扰 |
| Improved VQGAN | L2 归一化码本，余弦相似度分配 |
| RQ-VAE-Rotation | 用 rotation trick 替换 STE |
| SimRQ | 冻结码本，线性投影生成离散码 |
| RQ-OPQ | 残差量化 + 优化乘积量化 |
| RQ-Kmeans | 两阶段：协同对齐 + K-means 构建码本 |

### 评估指标

- **离线**：HitRate@K、NDCG@K（K∈{5,10}）、SID 组合熵 $\mathcal{E}_{\text{SID}} = -\sum_{\mathbf{s}} p(\mathbf{s})\log p(\mathbf{s})$
- **在线**：Completed Orders（CO）、GMV（GMV-S1/S2）、GPM（GMV per mille exposures）

### 实现细节

- 生成式推荐骨干：TIGER
- 离线：$L=3, K=256$；工业：$L=4, K=1024$
- 所有结果取 5 个随机种子平均
- 优化器：Adam（lr $3\times10^{-4}$，weight decay $1\times10^{-5}$），batch size 256
- Transformer：8 层，8 头，embedding dim 128，MLP hidden dim 512
- 超参数：$R=1$（离线），$R=2$（在线），$m_{\text{full}}=0.8$，$m_{\text{partial}}=0.5$

> **QuaSID 在所有指标上均取得最优结果**，top-K 排名质量平均提升 **5.9%**，同时 SID 组合熵最高（离散空间利用最充分）。

### HaMR 的即插即用效果

对多个基线方法添加 $\mathcal{L}_{\text{HaMR}}$ 后，NDCG@10 在 Beauty 上提升最高 **15.3%**，在 Toys 上提升最高 **20.1%**，熵提升约 0.1%–1.9%。

> **关键发现**：添加 HaMR 后基线的熵往往高于 QuaSID，但排名指标仍低于 QuaSID。说明 HaMR 提升了离散表示质量，而协同对比学习对下游任务对齐同样不可缺少，两者互补。

- **规模**：约 2000 万用户，5% 流量，持续 5 天
- $100vv$/$600vv$ 表示上线 48 小时内播放量 < 100/600 的冷启动视频

### 消融实验

| 模型 | Beauty HR@5 | Beauty NDCG@5 | Toys HR@5 | Toys NDCG@5 |
|------|-------------|---------------|-----------|-------------|
| QuaSID（完整） | **0.0277** | **0.0193** | **0.0266** | **0.0193** |
| w/o CVPM | 0.0263 | 0.0181 | 0.0264 | 0.0190 |
| w/o HaMR | 0.0254 | 0.0170 | 0.0261 | 0.0189 |

- **去掉 CVPM**：良性重叠被错误纳入排斥集，推开语义对齐的 item 对，性能下降
- **去掉 HaMR**：失去碰撞抑制能力，离散空间语义区分度下降

### 超参数敏感性分析（Amazon-Beauty）

| 超参数 | 最优值 | 规律 |
|--------|--------|------|
| $\lambda_{cl}$（协同对比权重） | 0.1 | 过大（0.5）会主导优化，干扰离散化 |
| $\lambda_{\text{full}}$（完全碰撞惩罚） | 0.2 | 过大会扭曲表示空间 |
| $\lambda_{\text{partial}}$（部分碰撞惩罚） | 0.1 | 过大会不必要地分离共享 token 的语义相关 item |

## 核心贡献总结

1. **提出 QuaSID 框架**：首个资质感知的 SID 学习框架，通过 HaMR 将低汉明距离 SID 重叠转化为编码器空间中的严重程度自适应几何边界约束。

2. **提出 CVPM**：通过掩码协议诱导的良性重叠（重复采样、构建的协同正样本）来净化碰撞监督，为排斥学习提供更干净的冲突集合。

3. **工业验证**：在快手电商平台大规模在线 A/B 测试中验证了 QuaSID，在排名 GMV-S2 提升 2.38%、冷启动召回 Completed Orders 提升最高 6.42%，证明了工业价值。

4. **即插即用**：HaMR 损失可无缝集成到多种 SID 学习框架中，作为通用的碰撞抑制插件。
