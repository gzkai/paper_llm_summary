## 研究背景与动机

### 语义 ID（SID）的作用

SID 是从多模态物品特征（文本、图像、音频、结构化元数据）中提炼出的紧凑离散编码，作为 ID-based 推荐和生成式推荐的统一抽象接口。相比传统 ID Embedding，SID 具备：
- **无词表爆炸**：固定长度 token 序列，避免 Embedding 表无限增长
- **跨系统一致性**：同一 SID 可在召回和排序中复用
- **生成式兼容**：SID 可直接输入 LLM/Seq2Seq 模型生成推荐

主流方法基于 **RQ-VAE**（残差向量量化变分自编码器），将连续 embedding 层次化量化为多个码本的离散 token 序列。

### 两大核心挑战

**挑战一：碰撞问题（Collision Problem）**

当大量物品被压缩到量化编码空间时，RQ-VAE 常出现码本利用不均或质心坍塌，导致语义无关的物品被映射到相同或高度相似的 SID 组合，产生**语义纠缠**，下游模型难以区分不同物品。

**挑战二：碰撞信号异质性（Collision-Signal Heterogeneity）**

关键洞察：**并非所有碰撞都同等有害！**

| 碰撞类型 | 来源 | 处理方式 |
|---------|------|---------|
| **有害碰撞** | 语义无关物品被迫共享 SID | 应强力排斥 |
| **良性重叠** | 同一物品的重复采样 | 不应排斥（会错误分离） |
| **任务一致正例** | 协同过滤构造的正对 | 不应排斥（会破坏任务对齐） |

现有方法（如 LETTER、HiD-VAE）对所有碰撞一视同仁地压制，会将良性对错误分离，引入虚假梯度，破坏训练稳定性。

## 核心方法：QuaSID 框架

QuaSID（**Qu**alification-**a**ware **S**emantic **ID** Learning）通过三个协同模块系统性解决上述问题。

### 整体架构

```
多模态特征 x_i
    ↓ 编码器 f_θ
连续 embedding z_i
    ↓ L 层残差向量量化
量化 embedding ẑ_i + SID s_i = [s_i^(1), ..., s_i^(L)]
    ↓ 解码器 h_φ
重建特征 x̂_i（重建损失）

同时：
  HaMR：批内低 Hamming 距离 SID 对 → 几何间距约束
  CVPM：掩码良性重叠对，净化排斥监督
  双塔对比学习：注入协同信号
```

**总损失**：

```math
\mathcal{L} = \mathcal{L}_{\text{rec}} + \mathcal{L}_{\text{rq}} + \mathcal{L}_{\text{HaMR}} + \mathcal{L}_{\text{cl}}
```

---

### 1. RQ-VAE 骨干网络

编码器将多模态特征映射为连续 embedding：

```math
\mathbf{z}_i = f_{\theta}(\mathbf{x}_i) \in \mathbb{R}^{d}
```

L 层残差量化，每层选最近邻码字：

```math
\mathbf{q}_i^{(l)} = \mathbf{c}^{(l)}_{s_i^{(l)}}, \quad
\mathbf{r}_i^{(l)} = \mathbf{r}_i^{(l-1)} - \mathbf{q}_i^{(l)}, \quad
\hat{\mathbf{z}}_i = \sum_{l=1}^{L}\mathbf{q}_i^{(l)}
```

SID 为离散 token 序列：$` \mathbf{s}_i = [s_i^{(1)}, s_i^{(2)}, \dots, s_i^{(L)}] \in \{1,\dots,K\}^L `$

骨干损失（重建 + 量化承诺）：

```math
\mathcal{L}_{\text{rq}} = \frac{1}{|\mathcal{B}|}\sum_{i\in\mathcal{B}}\sum_{l=1}^{L}
\Big(
\|\mathrm{sg}[\mathbf{r}_i^{(l-1)}]-\mathbf{q}_i^{(l)}\|_2^2
+ \beta\|\mathbf{r}_i^{(l-1)}-\mathrm{sg}[\mathbf{q}_i^{(l)}]\|_2^2
\Big)
```

其中 $`\mathrm{sg}[\cdot]`$ 为 stop-gradient，$`\beta`$ 控制承诺强度。

---

### 2. 协同双塔对比对齐

通过 **Swing** 算法从用户交互日志中构建物品-物品协同正对（trigger-target pair），注入行为语义信号：

给定批内归一化 trigger embedding $`\{\mathbf{e}_t^m\}`$ 和 target embedding $`\{\mathbf{e}_p^n\}`$，计算缩放相似度：

```math
\mathbf{S}_{m,n} = \frac{(\mathbf{e}_t^m)^{\top} \mathbf{e}_p^n}{\tau}
```

InfoNCE 对比损失（含 false-negative 掩码 $`M_{m,n}`$，排除批内同 ID 对）：

```math
\mathcal{L}_{\text{cl}} 
= - \frac{\lambda_{\text{cl}}}{B} \sum_{m=1}^{B} 
\log \frac{\exp(\mathbf{S}_{m,m})}
{\sum_{n=1}^{B} \mathbb{I}[M_{m,n}=1]\exp(\mathbf{S}_{m,n})}
```

---

### 3. CVPM：冲突感知有效对掩码

CVPM 通过掩码两类良性重叠对，净化排斥监督集合：

**协同正例掩码**（排除 Swing 构造的正对，否则排斥会与任务对齐目标矛盾）：

```math
\mathbf{M}^{\text{i2i}}_{ij} =
\begin{cases}
0, & (i \le B \wedge j=i+B) \lor (j \le B \wedge i=j+B) \\
1, & \text{otherwise}
\end{cases}
```

**同物品排除掩码**（排除同一底层物品 ID 的重复采样，避免将重复曝光视为碰撞）：

```math
\mathbf{M}^{\text{item}}_{ij} = \mathbb{I}\big[\mathrm{id}(i) \neq \mathrm{id}(j)\big]
```

组合掩码（Hadamard 积）：

```math
\mathbf{M} = \mathbf{M}^{\text{i2i}} \odot \mathbf{M}^{\text{item}}
```

仅对 $`\mathbf{M}_{ij}=1`$ 的对施加排斥，确保排斥仅针对真正的有害碰撞。

---

### 4. HaMR：Hamming 引导的间距排斥

**核心思想**：在同一 mini-batch 中，若两个不同物品的 SID Hamming 距离异常低，说明编码器空间中分离不足，应在连续 embedding 空间中施加几何间距约束。

定义批内 Hamming 距离矩阵和余弦距离矩阵（$`2B`$ 个实例，包含 trigger 和 target）：

```math
\mathbf{H}_{ij} = d_H(i,j) = \sum_{l=1}^{L} \mathbb{I}[s_i^{(l)} \neq s_j^{(l)}]
```

```math
\mathbf{D}_{ij} = d_c(i,j) = 1 - \mathbf{e}_i^\top \mathbf{e}_j
```

**按碰撞严重程度分类**（$`R`$ 为 Hamming 半径超参数）：

| 碰撞类型 | 条件 | 间距 | 惩罚强度 |
|---------|------|------|---------|
| **完全碰撞**（Full） | $`\mathbf{H}_{ij}=0`$ 且 $`\mathbf{M}_{ij}=1`$ | $`m_{\text{full}}`$（较大） | $`\lambda_{\text{full}}`$（较强） |
| **部分碰撞**（Partial） | $`0 < \mathbf{H}_{ij} \le R`$ 且 $`\mathbf{M}_{ij}=1`$ | $`m_{\text{partial}}`$（较小） | $`\lambda_{\text{partial}}`$（较弱） |

Hinge 损失（当余弦距离不足间距时惩罚）：

```math
\mathcal{L}_{\text{full}}(i,j) = \max\big(0, m_{\text{full}} - \mathbf{D}_{ij}\big)
```

```math
\mathcal{L}_{\text{partial}}(i,j) = \max\big(0, m_{\text{partial}} - \mathbf{D}_{ij}\big)
```

HaMR 总损失：

```math
\mathcal{L}_{\text{HaMR}}
= \frac{\lambda_{\text{full}}}{|\Omega_{\text{full}}|+\epsilon}
   \sum_{(i,j)\in\Omega_{\text{full}}}\mathcal{L}_{\text{full}}(i,j)
+ \frac{\lambda_{\text{partial}}}{|\Omega_{\text{partial}}|+\epsilon}
   \sum_{(i,j)\in\Omega_{\text{partial}}}\mathcal{L}_{\text{partial}}(i,j)
```

**几何含义**：对 $`\Omega_{\text{full}}`$ 中的对，零 hinge 损失意味着 $`d_c(i,j) \ge m_{\text{full}}`$，即在超球面上强制拉开角度间距，逐步重塑编码器空间，减少碰撞频率。

### 5. 工业部署流程

1. **离线周期训练**：基于 trigger-target 对生产速度，定期端到端训练 QuaSID（STE 直通估计器支持离散码分配的可微学习）
2. **SID 推理服务**：将 item_id 映射为 SID，对应新的 Lookup Table
3. **召回阶段应用**：
   - 传统判别式召回：SID 作为 cross feature 和 matching side feature
   - SID-based 生成式召回：直接用 SID 作为生成目标
4. **排序阶段应用**：SID 作为轻量语义信号增强现有特征集

## 实验结果

### 实验设置

| 数据集 | 规模 | 说明 |
|--------|------|------|
| **Amazon-Beauty** | 22,363 用户，12,101 物品，1,048,296 交互 | 5-core 过滤，leave-one-out 划分 |
| **Amazon-Toys** | 19,412 用户，11,924 物品，905,253 交互 | 同上 |
| **快手工业数据集** | 2000万+ 用户（5% 流量） | 多模态特征（文本+ASR+关键帧图像），MLLM 提取 embedding |

**评估指标**：HR@K、NDCG@K（K∈{5,10}）；SID 组合熵 $`\mathcal{E}_{\text{SID}} = -\sum_{\mathbf{s}} p(\mathbf{s}) \log p(\mathbf{s})`$（越大表示 SID 多样性越好）

**生成式推荐骨干**：TIGER（所有方法统一使用）

**码本配置**：离线 $`L=3, K=256`$；工业 $`L=4, K=1024`$

**基线**：RQ-VAE、GRVQ、Improved VQGAN、RQ-VAE-Rotation、SimRQ、RQ-OPQ、RQ-Kmeans

---

### 公开数据集主要结果

| 方法 | Beauty HR@5 | Beauty NDCG@10 | Beauty 熵 | Toys HR@5 | Toys NDCG@10 | Toys 熵 |
|------|------------|---------------|---------|----------|-------------|--------|
| RQ-VAE | 0.0225 | 0.0195 | 9.3075 | 0.0206 | 0.0180 | 9.3068 |
| RQ-Kmeans | 0.0254 | 0.0211 | 9.3793 | 0.0260 | 0.0213 | 9.3460 |
| **QuaSID** | **0.0277** | **0.0230** | **9.3901** | **0.0266** | **0.0225** | **9.3794** |

QuaSID 在所有指标上全面领先，**top-K 排序质量相比最强基线平均提升 5.9%**，同时 SID 组合多样性（熵）最高。

**熵与排序质量的相关性**：Pearson $`r \approx 0.65`$，Spearman $`\rho \approx 0.72`$（统计显著），但熵并非充分预测因子，任务语义对齐同样关键。

### HaMR 即插即用分析

将 $`\mathcal{L}_{\text{HaMR}}`$ 作为辅助目标添加到各基线（保持其他配置不变），验证其泛化性：

| 基线 | Beauty NDCG@10 提升 | Toys NDCG@10 提升 |
|------|-------------------|-----------------|
| RQ-VAE + HaMR | **+10.3%** | **+12.8%** |
| Improved VQGAN + HaMR | **+15.3%** | **+20.1%** |
| GRVQ + HaMR | **+12.3%** | **+16.8%** |
| RQ-OPQ + HaMR | **+12.5%** | **+1.0%** |
| RQ-VAE-Rotation + HaMR | +2.0% | +2.0% |
| SimRQ + HaMR | +4.1% | +5.3% |

HaMR 对所有基线均带来正向提升，NDCG@10 最高提升 **+20.1%**，熵提升 0.1%–1.9%，验证了其**模型无关、即插即用**的特性。

> 注：添加 HaMR 后基线的熵往往高于 QuaSID，但排序指标仍低于 QuaSID，说明 HaMR（提升离散表示质量）和对比任务监督（确保下游任务对齐）是互补的，缺一不可。

### 消融实验

| 变体 | Beauty HR@5 | Beauty NDCG@5 | Toys HR@5 | Toys NDCG@5 |
|------|------------|--------------|----------|------------|
| **QuaSID（完整）** | **0.0277** | **0.0193** | **0.0266** | **0.0193** |
| w/o CVPM | 0.0263 | 0.0181 | 0.0264 | 0.0190 |
| w/o HaMR | 0.0254 | 0.0170 | 0.0261 | 0.0189 |

- **去掉 CVPM**：良性重叠被错误纳入排斥集，引入虚假分离，降低性能
- **去掉 HaMR**：失去碰撞抑制，语义判别性下降，性能一致下滑

### 超参数敏感性分析（Amazon-Beauty）

| 超参数 | 最优值 | 规律 |
|--------|--------|------|
| $`\lambda_{\text{cl}}`$（对比权重） | 0.1 | 过小→协同信号不足；过大→主导优化，破坏离散化 |
| $`\lambda_{\text{full}}`$（完全碰撞权重） | 0.2 | 过小→惩罚不足；过大→过度排斥，扭曲表示空间 |
| $`\lambda_{\text{partial}}`$（部分碰撞权重） | 0.1 | 过小→局部码本拥挤；过大→不必要分离语义相关物品 |

## 核心贡献总结

1. **问题洞察**：首次系统性指出碰撞信号的异质性——并非所有 SID 重叠都有害，一刀切的碰撞抑制会引入虚假梯度

2. **HaMR（Hamming 引导间距排斥）**：将批内低 Hamming 距离 SID 重叠转化为编码器空间中的显式几何间距约束，按碰撞严重程度（完全/部分）自适应调整惩罚强度；**即插即用**，可无缝集成到任意 SID 学习框架

3. **CVPM（冲突感知有效对掩码）**：通过掩码同物品重复对和协同正例对，从碰撞监督中过滤良性重叠，提供更干净的排斥监督集合

4. **双塔对比对齐**：基于 Swing 协同对注入行为语义信号，确保 SID 与下游推荐目标对齐

5. **工业验证**：快手电商 2000万+ 用户规模验证，冷启动完成订单 **+6.42%**，排序 GMV **+2.38%**，已部署上线
